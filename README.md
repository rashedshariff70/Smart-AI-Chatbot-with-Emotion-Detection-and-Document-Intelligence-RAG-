"# Smart-AI-Chatbot-with-Emotion-Detection-and-Document-Intelligence-RAG-" 
🚀 NeuralMind AI

NeuralMind AI is a full-stack AI project that combines emotion detection, document-based Q&A (RAG), and a chatbot into one platform. It demonstrates how multiple AI features can work together in a single application.

✨ Features
Emotion Detection – Analyze text and identify emotions with sentiment and intensity
Document Q&A (RAG) – Upload PDF, DOCX, or TXT and ask questions
Chatbot – Multi-turn conversation with session memory
🛠️ Tech Stack
Python, Flask
HTML, CSS, JavaScript
PyPDF2, python-docx
⚙️ Setup
git clone https://github.com/your-username/neuralmind-ai.git
cd neuralmind-ai
pip install -r requirements.txt
python app.py

Open: http://127.0.0.1:5000

🔌 Note

This project uses a pluggable LLM design.
Integrate your own model (OpenAI, Groq, Ollama, etc.) in the backend to enable full AI functionality.

📡 API
/api/emotion – Emotion analysis
/api/upload – Upload document
/api/rag – Document Q&A
/api/chat – Chatbot
